# Hand & Finger Detection

A real-time **Hand & Finger Detection** project built using **Python**, **OpenCV**, and **MediaPipe**. The application detects hands from a webcam feed, tracks **21 hand landmarks**, and identifies raised fingers with high accuracy. This project provides a foundation for gesture recognition, touchless interfaces, and other computer vision applications.

---

## Features

- Real-time hand detection
- Finger counting
- Raised finger detection
- Multi-hand support
- Fast and lightweight implementation
- Easy to customize for gesture recognition

---

## Tech Stack

- Python
- OpenCV
- MediaPipe

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/Hand-Finger-Detection.git
cd Hand-Finger-Detection
```

### 2. Create a virtual environment (Optional)

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the project using:

```bash
python "Hand & Finger Detection.py"
```

A webcam window will open and start detecting hands and tracking finger movements in real time.
